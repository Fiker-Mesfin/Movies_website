# Project Title

**Movie Explorer**

## Project Description

Movie Explorer is a responsive movie discovery web application that allows users to browse and explore a wide collection of movies through third-party API integration. The application provides an attractive and user-friendly interface for discovering popular, trending, top-rated, and upcoming movies.

Instead of hosting movie content, the platform retrieves real-time movie information from external movie APIs, including posters, titles, release dates, ratings, genres, overviews, and other relevant details. Users can search for movies, browse different categories, and view detailed information about each title.

The application focuses on delivering a modern and visually appealing user experience while demonstrating the integration of REST APIs into a React application.

## Features

* Search movies by title
* Browse trending, popular, top-rated, and upcoming movies
* Display movie posters
* Favorities section
* Responsive design for desktop, tablet, and mobile devices
* Modern and intuitive user interface
* Fast loading and smooth navigation
* Real-time data fetched from external movie APIs

## Technologies Used

### Frontend

* React
* JavaScript
* HTML5
* CSS3
* React Router

### APIs

* Movie Database API

## Note

This application is intended for movie discovery only. It does not stream or play movies. Instead, it aggregates and displays movie information obtained through third-party APIs to help users explore and learn about different films.
